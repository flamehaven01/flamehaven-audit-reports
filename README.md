# flamehaven-audit-reports

Public archive of audit reports, review artifacts, and structured evaluation outputs from Flamehaven systems.

This repository stores generated reports from multiple systems, including:
- STEM-BIO-AI repository audits
- TOE-related audits
- RExSyn bio experiment audits
- other methodology and governance review artifacts

These materials are archival outputs, not source code releases.

## Structure

- `stem-bio-ai/` — repository audit outputs generated by STEM-BIO-AI
- `toe/` — TOE-related audit and evaluation artifacts
- `rexsyn-bio/` — RExSyn bio experiment audit artifacts

Each target should normally be stored under a dated folder, for example:

- `stem-bio-ai/yorkeccak-bio/2026-05-15/`
- `toe/example-target/2026-05-15/`
- `rexsyn-bio/example-experiment/2026-05-15/`

Typical contents may include:
- `report.html`
- `report.md`
- `report.json`
- `explain.txt`
- `report.pdf`

## Notes

These reports are evidence-surface or experiment-review artifacts.

They do not automatically imply certification, compliance, efficacy, or deployment approval.
